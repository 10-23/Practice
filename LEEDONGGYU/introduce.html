자기 소개

안녕하세요 React 프론트엔드 개발자 이동규입니다.

개발 역량

  • HTML로 웹 표준 및 사용성을 고려한 시맨틱 마크업 가능
  • CSS, Sass, CSS Module, Styled Component, Emotion, Tailwind, Mantine을 활용한 스타일링 가능
  • ES6 문법에 대한 이해를 기반으로 명령형 프로그래밍 JavaScript 코드 작성 가능
  • Express.js를 활용한 API 구축, 인증, 쿠폰 등 목업 서버 구현 가능
  • Git CLI, GitHub를 활용한 협업 가능
  • ESLint, Prettier를 활용한 코딩 컨벤션 유지 가능

개발 태도

  • 한번 맡은 업무는 끝까지 완수합니다.
  • 저는 개발 아이덴티디가 존재하기 때문에 더 나은 코드를 찾아 심사숙고하여 프로그래밍합니다.
  • 리더 역할을 맡은 경험이 있어서 경청과 설득에 일가견이 있으며, 커뮤니케이션의 중요성을 알고있습니다.
  • 열린마음으로 커뮤니케이션하기 떄문에 타인의 의견을 적극 수용합니다.

희망 사항

  • 개발 업무의 완성도를 높이기 위해 개발에만 전념합니다.
  • 스스로 개발 아이덴티디를 가지고 있기 때문에, 강압적인 환경에서의 코딩은 지양합니다.
  • 책, 강의, 컨퍼런스 등 최신 개발 기술을 습득하기 위한 금전적/환경적 제도 지원이 필요합니다.
